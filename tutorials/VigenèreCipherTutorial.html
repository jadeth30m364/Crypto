Vigenère Cipher

In the Vigenère Cipher, there is a keyword that is given. That keyword is the base of the code; in the grid below, there is an alphabet on both the horizontal and vertical axis, and the letters in the middle are the coded letters.

If there is the word, “MUG” and the keyword is “HAT” then in order to code using the Vigenère Cipher, you would first start with the “M” from “MUG.” You would find “M” on the horizontal axis. Then, you would take the first letter of “HAT” which is “H,” and you would find it on the vertical axis. Where the two rows/columns meet is the first letter, which if “T” in this case.

You can continue doing this with the rest of "MUG" to get, “TUZ” as the encrypted word.
