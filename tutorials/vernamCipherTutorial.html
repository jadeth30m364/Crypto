Vernam's Cipher

Vernam's cipher (used for the Lorenz machine) uses a keyword, like the vingere, but uses a different decoding and encoding method. Each letter or symbol (including spaces) in the cleartext and keyword is converted to a 6 digit binary number, and the keyword is repeated to fit the length of the cleartext. For example, the word "hi" with a keyword of "b" would become the following.
000111001000 (hi)
000001000001 (bb)
Then, an XOR operation is peformed on them. XOR returns 1 if both letters are the different, and 1 if they are the same. The above code would become:
000110001001 (gj)
To decrypt, the XOR operator is performed between the final text and key.
